# play-doudizhu-guide

一个用于归档和发布斗地主相关独立 HTML 页面的仓库。本仓库不针对任何特定网站或域名，仅作为斗地主玩法、规则、技巧等内容的静态页面集合。

## 项目简介

本项目旨在整理和分享与斗地主游戏相关的 HTML 页面。这些页面可能包含游戏规则说明、策略分析、牌型演示、学习笔记等，所有内容以独立 HTML 文件形式存放，便于直接浏览和离线使用。

## 目录说明

```
play-doudizhu-guide/
├── pages/          # 存放所有 HTML 页面
├── assets/         # 页面引用的资源文件（图片、样式、脚本等）
└── README.md       # 本文件
```

- **pages/**：主要页面目录，每个 HTML 文件为一个独立页面。
- **assets/**：页面所需的公共资源，按需组织。

## 页面归档说明

- 所有页面均为独立 HTML 文件，可直接在浏览器中打开。
- 页面之间无依赖关系，可单独查看。
- 页面内容可能包含交互式牌型演示、规则图解、策略卡片等。
- 归档页面不包含任何外部跟踪、广告或用户数据收集代码。

## 使用方式

1. 克隆或下载本仓库。
2. 打开 `pages/` 目录下的任意 HTML 文件。
3. 在浏览器中直接查看。

如需在线访问，可将仓库部署至任何静态文件托管服务（如 GitHub Pages、Netlify 等）。
